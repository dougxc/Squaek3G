Squawk Benchmarking

This document provides general information and links to a detailed description of the process used to benchmark the Squawk3G VM (15 Feb. 2005 Syntropy Build/Development Kit (BuildKit/DevKit) version).

Source Languages and Constraints

For the benchmarks, C and C++ without optimsations were compared with Java 1.4.2 in interpretter only mode and a Squawk3G installation prior to JIT compiler integration (i.e. also interpretter only).

Platforms

x86 (C, C++ and Java on Java 2 SDK 1.4.2_07 and Squawk3G)

  • Pentium Mobile Centrino (model 755, 2.0GHz), 1GB RAM, Windows XP Home

  • Pentium 4 Mobile (1.8 GHz), 512MB RAM, Windows XP Pro and Red Hat Linux 8.0

Atmel EB40a Embedded (C and Java on Squawk3G):

  • Atmel EB40a ARM7TDMI using C and Java on Squawk (Embedded platform).

Instructions

Instructions for how to run these benchmarks are available for x86 and the EB40a board.
